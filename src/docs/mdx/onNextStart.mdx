# onNextStart

`onNextStart` will be called at the beginning of slide transition.

`onNextStart` receive a callback with 2 arguments: `currentItem, nextItem`.


```
(currentItem, nextItem) => {/*...*/}
```

`currentItem, nextItem` is objects with these properties:

```
{
    item: {
        /* your item */
    },
    index: // the item's index number 
}
```
